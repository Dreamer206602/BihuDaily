# BihuDaily
高仿知乎日报

软件界面高仿官方知乎日报，采用MVP架构，网络请求使用Retrofit2.0+RxJava，图片加载使用Glide

部分资源文件来源于知乎日报的官方客户端，程序中所使用的所有API，都是通过破解知乎日报官方客户端得来，最终解释权归知乎所有。

## 实现功能

* 闪屏页图片获取
* 首页顶部轮播
* 下拉刷新，上拉加载
* 主题日报列表
* 缓存首页数据
* 新闻详情页
* 新闻评论展示
* 本地收藏
* 设置界面(无图模式，夜间模式，大字号，清除缓存)
